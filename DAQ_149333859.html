Peripheral Details

SSID: ICLR_DAQ

Password: rocketsAreCool!

Do not connect unless you are the test operator.

Integration Checklist

  • Remove DAQ protective cover.
  • Format SD card and plug into DAQ. The DAQ will NOT ignite or record data unless an SD card is detected.

  • Set switches to 'SAFE' position on the DAQ.

  • Plug in load cell as shown below. (Black/Yellow - GND, Red - 3V3, White - +Signal, Green - -Signal).

  • Plug in ematch into top right port (polarity independent) while keeping switch to SAFE. Do NOT insert ematch into engine.

  • Fit protective case back on.
  • Plug in provided LiPo battery into white JST connector, and verify unit turns on via power LED.
  • Connect to web interface and go to http://192.168.4.1/status endpoint.
  • Toggle ematch to 'ARM'  WITHOUT plugging it into the engine. Verify again that ematch does not trigger. Verify that continuity LED is ON (both be checked on status webpage). Toggle back to 'SAFE'.
  • Insert ematch into engine. Stand back a safe distance, and set safety switch to 'ARM'. Ematch should not trigger if status has been checked in prior step.
  • Connect to web interface http://192.168.4.1/pyro, and press button when ready to fire engine. Data logging will start as soon as ematch has been ignited.

Data Format

Data format is CSV. The columns (left to right) are:

  1. Ambient temperature
  2. Thermo 1 temperature
  3. Thermo 2 temperature
  4. Thermo 3 temperature
  5. Thermo 4 temperature
  6.  Pressure transducer 1
  7.  Pressure transducer 2
  8.  Pressure transducer 3
  9.  Pressure transducer 4
  10.  Pressure transducer 5
  11.  Differential input 1
  12.  Differential input 2
  13.  Differential input 3
  14.  Differential input 4
  15.  Milliseconds since turned on
  16.  Ematch status


The approximate derivation for force:

Load cell output = 20mV/V = 20mV * (3.3 V operating voltage) * (Gain of 4) at 300 kg (2940 N). 

= 264 mV output at 2940 N

= 8.97959184e-5 V/N

At 1.2 V, there are 2^23 counts => 6990506.67 counts/V

= 627.718967 counts/N conversion factor.

=> F = (counts)/627.718967

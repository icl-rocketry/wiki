ICL Rocketry Wiki : Test fire procedure checklist (solid motor)

Created by Sokolowski, Rafal on Feb 02, 2020
[] Assemble and adjust stand to fit casing.
[] Carry stand to testing position and stake into ground -> if ground is moist try to place it on waterproof cover
[] Check to ensure nothing is loose; make sure load cell rods don’t have loose nuts.
[] Connect electronics; daq, battery, do not connect e match (Checklist for daq is under payload).
[] Test daq -> connect to web interface and dry run arming, firing and safing (Payload checklist).
[] Remove daq as far away as possible, place on a waterproof cover. 
[] Refer to payload checklist to test ematch itself.
[] Switch off ematch system. 
[] Insert motor into casing, secure and insert ematch with black powder.
[] Switch system back on and fire when ready.

Details for Hot Test of Static Fire Test Rig

  • we did not inform EARS we were planning on doing this test prior to showing up. 
  • When we arrived, we were told that the test rig must be vertical
  • We modified the rig by mounting the plate by clamping it in two tripods
  • We had this vetted by the range safety officers
  • Performed the test

Challenges:

  • Changing the motor grain tube was very time consuming
  • The rig is not designed to be vertically mounted, even though we managed. Some thoughts on the ease of making it adaptable between both options is warranted.
    • Vertical to ensure the (motor + test sled)/(just motor) don't go flying
  • DAQ did not connect to the phone, but by overriding some features we managed to make it work. 
  • Noise level on DAQ prevented us from immediately processing the data and seeing a graph that worked. It was found later on.
  • No easy way to verify the working DAQ, or seeing a live plotting. 
  • A post-processing python script should be made before the experiment so that the results can be immediately analysed. 
  • Remote ignition using the ESP was not allowed. Used the RSO's ignition leads instead. Not problematic for this test, but for our tests we would like to know the time delay between hitting the button and the motor starting

Clarification on process for DAQ box

Details on Parachute

At the end we spent an hour or more discussing the recovery system with Andy and John. A few learning points:

  • No chute release - no one seems to find them reliable
  • Length of shock cords to be balanced to form the V. The nose cone end must lie further down than the lower part of the rocket under the drogue chute. Then the main gets deployed off the nose cone end. 
  • Parachute packaging tips (essentially just stuff it in, but shroud lines and shock cords can be stored on the outside of the bag using some elastic cord.
  • Ejection caps using centrifuge tubes in brass pipes. Reusable and reliable.
    (Embed the ejection charge in the middle of the centrifuge tube that has bp filled in it, and with wadding at the top to hold the BP in place so it can't shake around). 
  • AV bay with dual rocket split is a classic method
  • Not all nylon screws are made equal. Every manufacturer and every batch can be different. Ground tests must be done using  screws from the same batch as those that will be flying.
  • Parachute has an apex, and the drogue attaches there to literally pull the main out. 
  • Direction of ejection charge going off isn't important - ie. if the ejection effectively blows and stuffs the parachute deeper in a body tube that isn't strictly bad.
